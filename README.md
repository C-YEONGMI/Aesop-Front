# Aesop Front

Frontend application for the Aesop renewal project.

## Stack

- React
- Vite
- Redux Toolkit
- React Router v6
- Tailwind + shadcn/ui

## Run

```bash
npm install
npm run dev
```

## Environment

Copy `.env.example` to `.env.local` and set the frontend values you need.

The app expects the backend API at `VITE_API_BASE_URL`.
